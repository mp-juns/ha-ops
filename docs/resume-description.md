- Integrated occupancy sensors, camera snapshots, and LLM Vision to classify room-level activities
- Structured daily activities such as cooking, eating, computer work, sleep, and phone usage into Home Assistant helper entities
- Implemented room cleanliness scoring using PTZ camera presets and LLM Vision analysis
- Configured Wake-on-LAN and SSH shell commands for remote PC/WSL environment control
- Used History Stats and helper entities to accumulate lifestyle data and generate daily reports
